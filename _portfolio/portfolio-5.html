Project Summary

To help my sister with her wedding, I built and deployed a responsive wedding website to GitHub Pages. The site is available at khaeby.github.io, and the source code is on GitHub.

I started from a GitHub template (shyamjos/wedding-website), forked it, and customized the HTML and CSS based on my sister's feedback. I ensured the site looks good on both desktop and mobile, added sections for event details, RSVP functionality, and a photo gallery, and updated the favicon and social preview thumbnails.

Technologies Used

  • Web Development: HTML, CSS, JavaScript
  • Version Control: GitHub
